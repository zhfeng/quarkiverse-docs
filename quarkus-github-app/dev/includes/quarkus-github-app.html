Configuration property fixed at build time - All other configuration properties are overridable at runtime

Configuration property

Type

Default

quarkus.github-app.app-id

The numeric application id provided by GitHub.

string

required

quarkus.github-app.app-name

The GitHub name of the application. Optional, only used for improving the user experience.

string

quarkus.github-app.private-key

The RSA private key.

PrivateKey

required

quarkus.github-app.webhook-secret

The webhook secret if defined in the GitHub UI.

string

quarkus.github-app.webhook-proxy-url

The Smee.io proxy URL used when testing locally.

string

quarkus.github-app.debug.payload-directory

A directory in which the payloads are saved.

path
